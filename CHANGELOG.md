# Changelog

All notable changes to DeepPaperNote will be documented in this file.

The format is intentionally lightweight for now.

## Unreleased

### Added

- Codex-first DeepPaperNote skill scaffold
- deterministic preprocessing pipeline for single-paper note generation
- model-facing synthesis bundle workflow
- Zotero-first helper flow
- placeholder-first figure planning
- Obsidian folder-per-paper output layout
- note linting for structure and mixed-language prose
- bilingual README (`README.md` and `README.zh-CN.md`)

### Changed

- shifted the architecture toward `model-first` paper understanding
- standardized figure placeholders to a stable callout format
- moved image output into paper-local `images/` folders

### Notes

- The project is still in early public-facing preparation.
- Packaging, tests, CI, and public examples are still being added.
